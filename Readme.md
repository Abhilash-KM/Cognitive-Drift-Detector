Cognitive Drift Detector

An AI system that models human reasoning patterns and detects cognitive drift using machine learning and deep learning techniques.

This project analyzes how reasoning style changes over time and detects unusual cognitive patterns.

---

Features

- Transformer-based text embeddings
- Statistical drift detection
- Neural autoencoder anomaly detection
- Reasoning pattern clustering
- Mode-switch detection
- Time-based cognitive analysis
- Real-time reasoning analysis
- Streamlit web interface

---

Technologies Used

- Python
- PyTorch
- Sentence Transformers
- Scikit-learn
- Pandas
- Matplotlib
- Streamlit

---

Machine Learning Methods

1. Transformer Embeddings
Reasoning responses are converted into vector embeddings using Sentence Transformers.

2. Statistical Drift Detection
Cosine similarity measures deviation from baseline reasoning.

3. Neural Drift Detection
Autoencoder neural network detects anomalous reasoning patterns.

4. Behavioral Clustering
KMeans clustering identifies different thinking modes.
